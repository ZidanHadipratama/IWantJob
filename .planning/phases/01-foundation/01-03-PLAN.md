**Step 1: Initialize Plasmo project.**
Run `npm create plasmo@latest -- --with-tailwindcss` in the extension/ directory (or create manually if the CLI doesn't support the flag). The project must be named "jobpilot-extension" with displayName "JobPilot".

If initializing manually:
- Create extension/ directory
- Run `npm init -y`, then install: plasmo, react, react-dom, @types/react, @types/react-dom, typescript, tailwindcss, postcss, autoprefixer
- Create tsconfig.json with Plasmo-compatible settings (strict: true, jsx: "react-jsx", moduleResolution: "bundler")

**Step 2: Configure package.json manifest.**
Ensure package.json contains:
```json
{
  "name": "jobpilot-extension",
  "displayName": "JobPilot",
  "version": "0.0.1",
  "description": "AI-powered job application assistant",
  "manifest": {
    "permissions": ["storage", "sidePanel", "tabs", "activeTab"],
    "side_panel": {
      "default_path": "sidepanel.html"
    }
  }
}
```

**Step 3: Configure Tailwind.**
- tailwind.config.js: content paths include `./src/**/*.{tsx,ts,jsx,js}`
- postcss.config.js: standard tailwindcss + autoprefixer plugins
- src/style.css: `@tailwind base; @tailwind components; @tailwind utilities;`

**Step 4: Create side panel entrypoint.**
CRITICAL: The file MUST be `src/sidepanel.tsx` (flat at src/ root) -- NOT `src/sidepanel/index.tsx`. Plasmo requires flat entrypoints.

src/sidepanel.tsx:
- Import `./style.css` for Tailwind
- Import the 4 tab components from `./components/sidepanel/`
- Define type Tab = "fill-form" | "resume" | "tracker" | "chat"
- useState<Tab>("fill-form") for active tab
- Tab bar: 4 buttons in a flex row with border-b. Active tab gets border-b-2 border-blue-500 text-blue-600. Inactive gets text-gray-500 hover:text-gray-700.
- Main content area: flex-1 overflow-auto p-4. Conditionally render the active tab's component.
- Container: flex flex-col h-screen bg-white

**Step 5: Create 4 tab components.**
Each in src/components/sidepanel/:
- FillForm.tsx: `export default function FillForm() { return <div className="text-center text-gray-400 mt-8">Fill Form -- coming in Phase 5</div> }`
- Resume.tsx: same pattern with "Resume" text
- TrackerTable.tsx: same pattern with "Tracker" text
- Chat.tsx: same pattern with "Chat" text

**Step 6: Create popup.**
src/popup.tsx:
- Import ./style.css
- Render a small card (w-64 p-4) with:
  - "JobPilot" heading (text-lg font-bold)
  - "Ready" status text (text-sm text-green-600)
  - "Open side panel for full interface" helper text (text-xs text-gray-400)

**Step 7: Create options page shell.**
src/options.tsx:
- Import ./style.css
- Render a centered container (max-w-2xl mx-auto p-8) with:
  - "JobPilot Settings" heading
  - "Configuration options coming in Phase 2" placeholder text

**Step 8: Verify build.**
Run `npm run dev` or `npm run build` and check that:
- build/chrome-mv3-dev/ directory is created
- manifest.json in build output contains "side_panel" key
- No TypeScript errors

Frequently asked questions

Do I need to be technical to use the open-source version?
You need basic comfort with Docker and terminal commands. The setup involves running docker compose up, creating a free Supabase project, and loading the extension in Chrome. The README walks through every step. If that feels like too much, the Pro plan will handle all of it for you.
Which AI providers are supported?
OpenAI, Anthropic (Claude), Google Gemini, DeepSeek, and Ollama for local models. You can even use different models for different features — for example, GPT-4o for resume tailoring and a cheaper model for form filling. The Pro plan includes built-in AI so you don't need any API key.
Is my data safe? Where is it stored?
With the open-source version, all data lives in your own Supabase database. API keys are stored in your browser's local storage and passed per-request — the backend never stores them. There is no telemetry, no analytics, and no logging of your content. The Pro plan uses managed infrastructure but maintains the same privacy-first approach.
Does the extension auto-submit applications?
No. The workflow is intentionally draft-first. AI generates answers, you review and edit them, then you decide whether to autofill and submit. Nothing saves to your tracker until you explicitly approve it. You stay in control of every application.
What job sites does it work on?
IWantJob uses Mozilla Readability for JD extraction and standard DOM analysis for form fields, so it works on most job sites without site-specific adapters. It handles standard HTML forms, custom combobox widgets, and even application forms hosted inside iframes. Some highly custom or JavaScript-heavy application portals may need manual field entry.
What's the difference between Open Source and Pro?
Same core features. The difference is setup and convenience. Open Source means you run the backend yourself, bring your own AI key, and manage your own database. Pro means we handle all infrastructure — you just install the extension and start applying. Pro also includes optimized AI prompts and priority support.
